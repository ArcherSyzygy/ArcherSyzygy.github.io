# 解析
1. 利用异或运算的性质
   1. x ^ x = 0
   2. x ^ 0 = x
2. 对所有数字异或运算
   1. $x_1$  ^ $x_2$ ^ …… ^ $x_n$ = 只出现过一次的数 
3. 程序
```cpp
int singleNumber(vector<int>& nums) {
     int res = 0;
     for(auto x : nums)
         res ^= x;
     return res;
}
```
