# Jekyll网站的基础结构
```
├── _config.yml 
├── _includes    
├── _layouts
├── _posts
├── _site
├── 404.html
├── about.md
├── archive.html
├── CNAME
├── css
├── feed.xml
├── Gemfile
├── Gemfile.lock
├── images
├── index.html
├── js
├── Rakefile
├── README.md
└── tags.html
```
很复杂看不懂是不是，不要紧，你只要记住其中几个OK了

- _config.yml 全局配置文件
- _posts	放置博客文章的文件夹
- img	存放图片的文件夹

其他的想继续深究可以[看这里](https://www.jekyll.com.cn/docs/structure/)
